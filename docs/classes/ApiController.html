    Class ApiController

    Controller class that handles HTTP requests for Midnight blockchain operations.

    This controller serves as the interface between Express routes and the FbNightApiService, handling various operations including address allocation, claims management, scavenger hunt challenges, and token redemption.

    ApiController

    Example

    const apiService = new FbNightApiService(config);
    const controller = new ApiController(apiService);

    app.get('/allocation/:vaultAccountId/:chain', controller.checkAddressAllocation);
